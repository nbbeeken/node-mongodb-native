Class Db

The Db class is a class that represents a MongoDB Database.

example

const { MongoClient } = require('mongodb'); // Connection url const url = 'mongodb://localhost:27017'; // Database Name const dbName = 'test'; // Connect using MongoClient MongoClient.connect(url, function(err, client) { // Select the database by name const testDb = client.db(dbName); client.close(); });
